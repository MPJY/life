#### 游戏玩法

游戏的玩法是:您扮演一位刚出社会的小伙子，应聘工作是您最好的选择。

应聘一份工作，不断升值加薪(还可以摸鱼哦,被发现的话,奖金扣100)，这期间您还需要不断经受一切可能发生的挫折。

如:因工作不好被克扣工资、因欺负员工被开除等等。完成这一切，您也可以变为老板，享受人生！
